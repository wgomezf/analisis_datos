En el curso de Análisis de Datos (segunda parte) impartido en el Cinvestav Tamaulipas, se estudiarán técnicas relevantes de aprendizaje supervisado para clasificar automáticamente objetos físicos de muchas dimensiones en categorías o clases conocidas. Estos métodos son útiles en muchas áreas de aplicación, como la minería de datos, el análisis y reconocimiento de documentos, la lingüística computacional, las ciencias forenses, la biometría, la bioinformática y el diagnóstico de enfermedades, entre otras. El curso está dividido en tres unidades: clasificación bayesiana, clasificadores lineales y redes neuronales artificiales.
